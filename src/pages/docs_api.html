Docs : API

Access Userbase from any context via REST endpoints exposed by the Admin API. To access the Admin API, you need to get an Access Token from the Account section in your Admin panel.

Users

Use these APIs to authenticate your user, and retrieve or modify your users' metadata.

  • Verify Auth Token
  • Get User
  • Update User
  • List Users
  • List Database Users

Apps

Use these APIs to retrieve apps' metadata.

  • Get App
  • List Apps
