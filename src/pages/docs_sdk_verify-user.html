Docs : SDK : Verify User

The Verify User API lets you verify another user using their verification message.

    
      userbase.verifyUser({ verificationMessage }).then(() => {
        // user verified
      }).catch((e) => console.error(e))

Notes

  • Users are verified by their username. If a user changes their username, they must be reverified.

Errors

  • ParamsMustBeObject
  • VerificationMessageMissing
  • VerificationMessageMustBeString
  • VerificationMessageCannotBeBlank
  • VerificationMessageInvalid
  • VerifyingSelfNotAllowed
  • UserNotSignedIn
  • TooManyRequests
  • ServiceUnavailable
